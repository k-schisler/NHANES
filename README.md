# NHANES
NHANES Dataset Investigations using RStudio

# Birth of the Repository
Initially started as a final project for a LSC 540: Statistics for Biological Data Science 1 course at Arizona State University. The project was to come up with a research question or testable question that can be answered using the techniques learned in class. My research question I chose was "can your marital status be used to predict your heart health, where pulse, blood pressure, and cholesterol act as indicators for heart health?"

# Beyond the Classroom
I've expanded the project past the confines of the class, using the dataset to explore functions in RStudio and investigate other relationships that I could find to practice statistical skills, R code, and presentation of data. 
I've used this dataset to explore the following concepts, R codes, etc:
- Libraries/Packages:
  - NHANES
  - ggplot2
      - Histograms
      - Boxplots
      - Violin Plots
      - QQ Plots
  - dplyr
  - mosaic
- Relationships/Research Questions:
  - Marital Status and heart health
    -  Pulse, Systolic Blood Pressure, Diastolic Blood Pressure, Pulse Pressure, Direct Cholesterol, Total Cholesterol
      - Filtered data to those between 20 and 55 years.
      - Checked for relationship with gender - is it different between them?
